## Usage

1.

    $ cd ~/

    $ git clone https://github.com/tuxslack/mako.git

    $ cd mako/

    $ mkdir -p $HOME/.config/mako/

    $ mv -i config $HOME/.config

    Root: mv -i usr /
  
    $ alternador-de-tema-mako.sh
